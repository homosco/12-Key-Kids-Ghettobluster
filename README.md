# 12 Key Kids-GhettoBluster
Arduino MP3 player for kidz. (Inspired by HÃ¶rbert which can only play WAV and 9 albums and has no display)
No text for small kidz -> TFT to display a picture of the album.
Buttons:
- Track Fwd
- Track Bwd
- 9 Albums
- Play/Pause

## Hardware
 - Arduino Uno R3
 - Adafruit Music Maker Shield
 - Adafruit PowerBoost 500 Charger - Rechargeable 5V Lipo USB Boost @ 500mA+
 - LiPo Akku  (3.7 V, 2 mm JST)
 - 12 Button Keypad
 - any Micro SD card
